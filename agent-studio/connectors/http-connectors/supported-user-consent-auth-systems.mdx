Below is a list of systems that we have validated work with our implementation of OAuth 2.0 authorization code grant types. Just because a system is not on here does not mean it will not work; it just means that we have not seen any customer or internal implementations of it. If you have implemented a User Consent Auth use case with a system that is not listed below, please edit this document and add the system.

Supported Systems:

* GitHub
* Workday
* Coupa
* Google Calendar
* Google Drive
* Slack
* MS Graph
* Salesforce
* Zoom
