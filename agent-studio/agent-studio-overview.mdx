# Agentic Frameworks

Agent Studio lets you connect AI assistants to your business systems & content. There are three **Agentic Frameworks** that you can build on.

<CardGroup style="grid-template-columns:repeat(3, 1fr)">
  <Card title="Content GatewayIndex your content for semantic search experiences." href="/docs/content-gateway" />

  <Card title="Conversational AI AgentsExecute real-time actions to create, modify, read, & remove data from business systems." href="/docs/conversational-agents" />

  <Card title="Ambient AI AgentsSet up Ambient AI agents that run in the background" href="/docs/ambient-agents" />
</CardGroup>

These agentic frameworks are interoperable on the Moveworks platform, so you can build experiences that span multiple patterns.
