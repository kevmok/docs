## Overview

Moveworks provides several APIs for integrating with the platform:

### Core APIs (Hosted by Moveworks)

<CardGroup cols={2}>
  <Card title="Events API" icon="bolt">
    Send messages and events to Moveworks on behalf of users
  </Card>
  <Card title="Conversations API" icon="comments">
    Programmatically manage conversation threads and messages
  </Card>
  <Card title="Data API" icon="chart-bar">
    Export analytics data for conversations, interactions, and users
  </Card>
  <Card title="Webhook Listeners" icon="webhook">
    Send webhooks to Agent Studio listeners for external triggers
  </Card>
</CardGroup>

### Gateway APIs (Customer-Hosted)

These APIs are implemented by you to integrate custom systems with Moveworks:

<CardGroup cols={2}>
  <Card title="Content Gateway" icon="file">
    Integrate custom content systems with Copilot Search
  </Card>
  <Card title="Identity Gateway" icon="user">
    Connect custom identity sources with Moveworks
  </Card>
  <Card title="Knowledge Gateway" icon="book">
    Make custom knowledge bases available through the AI assistant
  </Card>
  <Card title="Forms Gateway" icon="rectangle-list">
    Integrate custom forms and service catalogs
  </Card>
</CardGroup>

## Authentication

All API requests require authentication using an API key. Include your API key in the `Authorization` header as a Bearer token:

```bash
curl -X GET "https://api.moveworks.ai/..." \
  -H "Authorization: Bearer YOUR_API_KEY"
```
